如何部屬
  1. 打開魔獸爭霸三三淬鍊重生：
    選單→選項→輸入裝置，將預設按鍵設定改成「網格」，儲存設定後關閉魔獸爭霸遊戲程式。
  2. 前往設定檔路徑「War3Preferences.txt」：
    C:\Users\你的使用者名稱\Documents\Warcraft III
  3. 設定你的熱鍵：
    你可以利用本程式，依照自己的喜好設定「背包」與「技能」的熱鍵。只需要點擊輸入框，再按下想要設定的按鍵，程式就會自動記錄你的設定。
    請記住，所有的按鍵都必須定義。按鍵的排列對應遊戲內的按鍵位置，你可以參考下圖。
  4. 產生設定檔：
    按下產生設定檔的按鈕後，瀏覽器會自動下載「War3Preferences.txt」檔案，請將他複製到第二步驟打開的路徑覆蓋過原本的檔案即可。
  5. 魔獸爭霸三三淬鍊重生，享受你的新熱鍵吧！
